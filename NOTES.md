# NOTES

## Access to NS

### Using Typescript

```typescript
import { NS } from "@ns";
```

### Using embedded `nano`

Default, annotation in JSDoc style

```typescript
/** @param {NS} ns **/
```

### Availmable in VSCode through `tsc`

Default, annotation in JSDoc style

```typescript
/** @param {import("../NetscriptDefinitions.d.ts").NS} ns */
```
